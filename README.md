# Shopify Storefront API demo app

by Ira Herman

![screenshot](https://i.imgur.com/rYoJ3Y5.png)

# Technologies Used:

* ReactJS
* Tailwind CSS
* Shopify Storefront API
* GraphQL (via Axios HTTP POST request)

# Installation Instructions:

* `npm install`
* Replace the `API_KEY` in App.js with your own

# Run it:

* `npm start`
